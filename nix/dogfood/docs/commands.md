# Commands

- **`motd`** — print the static welcome banner; also available as `help` and `?`.
- **`menu`** — fuzzy-filter the configured command catalogue.
- **`menu list`** — print the grouped command table and exit.
- **`docs`** — open the Markdown pages declared under `prelude.docs.pages`.
- **`nix flake check`** — build packages and run repository checks.
