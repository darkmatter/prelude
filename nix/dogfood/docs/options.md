# Options

## `help`, `?`

Reprint the MOTD welcome banner.

## `menu`, `m`

Open the interactive command picker. Pass a command name or key to run it directly.

## `docs`, `d`

Open this viewer. Digits jump between pages; `j` and `k` scroll the viewport.

## `prelude.theme`

Choose a named palette such as `phosphor`, `prelude`, or `nord`. Override individual tokens with `prelude.palette`.

## `prelude.colorProfile`

Use `auto`, `truecolor`, or `ansi256`. Force true color when terminal detection is wrong.
