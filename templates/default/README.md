# My Prelude-powered devshell

Generated by `nix flake init -t github:darkmatter/prelude#default`.

## Usage

```sh
nix develop       # enter the devshell (MOTD prints automatically)
menu              # interactive command picker
docs              # Markdown docs viewer
x <name>          # run a command by catalogue key
```

Add project commands under `prelude.commands` and docs pages under
`prelude.docs.pages` in `flake.nix`.
