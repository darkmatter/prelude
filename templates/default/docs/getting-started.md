# Getting started

Welcome to your Prelude-powered devshell.

## Commands

- Run `menu` to search Prelude's navigation and your configured project commands.
- Run `docs` to browse these pages.
- Run `x <name>` to run a command directly (e.g. `x dev`).

## Next steps

Add project commands under `prelude.commands` in `flake.nix`, and add more
Markdown pages under `prelude.docs.pages` to grow this docs viewer.
